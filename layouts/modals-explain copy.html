Explication - Layout 1 (2 colonnes égales)

Structure du Layout 1

Ce layout présente une structure simple et classique avec deux colonnes de taille égale dans le contenu principal.

Composants

  • Header : Banneau supérieur qui s'étend sur la largeur totale (col-md-12)
  • Contenu : Deux colonnes de 6 unités chacune (col-md-6), prenant chacune 50% de la largeur
  • Footer : Pied de page qui s'étend sur la largeur totale (col-md-12)

Système de Grille

Basé sur la grille 12 colonnes de Bootstrap :

  • Chaque colonne de contenu = 6/12 = 50% de la largeur disponible
  • Le point de rupture "md" s'applique à partir de 768px de largeur d'écran
  • Sur les petits écrans (<768px), les colonnes se transforment en colonnes de 100% de largeur

Styles Appliqués

  • Header et Footer : Fond gris (#2f2f2f), hauteur minimale
  • Colonnes de contenu : Fonds en gris clair, hauteur minimale de 200px
  • Alternance de couleurs pour mieux différencier les colonnes

Cas d'Usage

Ce layout est idéal pour :

  • Les pages avec deux sections de contenu égales
  • Les comparaisons côte à côte (avant/après, produit A vs produit B)
  • Les pages d'accueil avec deux colonnes de contenu
  • Les mises en page simples et équilibrées
Explication - Layout 2 (3 colonnes inégales)

Structure du Layout 2

Ce layout présente le classique "Sidebar + Content" avec une barre latérale de navigation et un contenu principal large.

Composants

  • Header : Banneau supérieur plein écran (col-md-12)
  • Sidebar : Navigation latérale gauche (col-md-3, soit 25% de la largeur)
  • Content : Zone principale de contenu (col-md-9, soit 75% de la largeur)
  • Footer : Pied de page plein écran (col-md-12)

Système de Grille

Distribution 3/9 des 12 colonnes :

  • Sidebar = 3 colonnes (25%)
  • Content = 9 colonnes (75%)
  • Tous les éléments (Header/Footer/Sidebar/Content) s'alignent sur les mêmes points de grille

Styles Appliqués

  • Sidebar : Fond gris clair (#d4d4d4) pour la distinguer
  • Content : Fond blanc ou très clair
  • Header/Footer : Fond gris foncé pour délimiter les zones

Cas d'Usage

Ce layout est parfait pour :

  • Les blogs avec navigation latérale
  • Les sites de documentation
  • Les tableaux de bord avec menu de navigation
  • Les pages produit avec catégories latérales
  • Les interfaces d'administration
Explication - Layout 3 (Content + Sidebars)

Structure du Layout 3

Un layout asymétrique avec un contenu principal large à gauche et une barre latérale droite subdivisée en deux sections.

Composants

  • Header : Banneau supérieur (col-md-12)
  • Content Principal : Colonne gauche large (col-md-7, soit 58% de la largeur)
  • Sidebar Supérieure : Première section de la colonne droite (col-md-5)
  • Sidebar Inférieure : Deuxième section de la colonne droite (col-md-5)
  • Footer : Pied de page (col-md-12)

Système de Grille

Distribution 7/5 avec sous-grille dans la colonne droite :

  • Content = 7 colonnes (58%)
  • Sidebar droit = 5 colonnes (42%), contenant deux rangées imbriquées
  • Les sous-éléments du sidebar utilisent col-md-12 pour remplir leur parent

Styles Appliqués

  • Content Principal : Fond blanc avec hauteur minimale
  • Sidebars : Fonds en gris pour les distinguer
  • Alternance de couleurs entre sidebar-top et sidebar-bottom

Cas d'Usage

Idéal pour :

  • Les blogs avec barre latérale droite
  • Les articles avec widgets connexes
  • Les pages produit avec filtres et informations supplémentaires
  • Les portails d'information avec contenu principal et sections annexes
Explication - Layout 4 (3 colonnes égales)

Structure du Layout 4

Un layout symétrique et équilibré avec trois colonnes de taille égale, flanquées de deux sidebars.

Composants

  • Header : Banneau supérieur (col-lg-12)
  • Sidebar Gauche : Navigation ou widgets (col-lg-4)
  • Content Principal : Contenu central (col-lg-4)
  • Sidebar Droit : Contenu additionnel (col-lg-4)
  • Footer : Pied de page (col-lg-12)

Système de Grille

Distribution égale 4/4/4 des 12 colonnes :

  • Chaque colonne = 4 colonnes Bootstrap (33,33% de la largeur)
  • Utilise le point de rupture "lg" pour les écrans ≥992px
  • Sur les écrans plus petits, les colonnes s'empilent

Styles Appliqués

  • Sidebars : Fond gris (#d4d4d4) avec hauteur minimale
  • Content : Fond blanc ou très clair
  • Header/Footer : Fonds sombres pour délimitation

Cas d'Usage

Parfait pour :

  • Les galeries avec contenu central et widgets latéraux
  • Les tableaux de bord équilibrés
  • Les sites avec navigation double latérale
  • Les pages avec trois sections de contenu égales
  • Les interfaces de comparaison
Explication - Layout 5 (3 colonnes inégales)

Structure du Layout 5

Un layout asymétrique avec un contenu principal au centre (plus grand) et deux sidebars plus petites de chaque côté.

Composants

  • Header : Banneau supérieur (col-lg-12)
  • Sidebar Gauche : Navigation (col-lg-3, soit 25%)
  • Content Principal : Contenu central (col-lg-6, soit 50%)
  • Sidebar Droit : Widgets/info (col-lg-3, soit 25%)
  • Footer : Pied de page (col-lg-12)

Système de Grille

Distribution 3/6/3 des 12 colonnes :

  • Sidebars = 3 colonnes chacun (25%)
  • Content = 6 colonnes (50%)
  • Équilibre : sidebars plus petits, contenu dominant

Styles Appliqués

  • Content Central : Fond blanc/très clair, position visuelle dominante
  • Sidebars : Fonds gris pour les différencier
  • Tous avec hauteurs minimales définies

Cas d'Usage

Idéal pour :

  • Les blogs avec sidebars de widgets
  • Les articles avec navigation et contenu connexe
  • Les pages d'accueil avec contenu principal et sidebars
  • Les interfaces avec contenu principal central
  • Les magazine layouts
Explication - Layout 6 (3 colonnes complexes)

Structure du Layout 6

Un layout complexe avec sidebars gauche et droit, et un contenu central avec une sous-grille comportant des sections imbriquées.

Composants

  • Header : Banneau supérieur plein écran
  • Sidebar Gauche : Navigation (col-3, soit 25%)
  • Contenu Central : Sous-grille complexe (col-md-7, soit ~58%)
    • Section supérieure : Zone de contenu principal
    • Section inférieure : Deux colonnes de contenu
  • Sidebar Droit : Widgets compacts (col-2, soit ~17%)
  • Footer : Pied de page plein écran

Système de Grille

Distribution 3/7/2 avec sous-grille imbriquée :

  • Sidebar gauche = 3 colonnes (25%)
  • Content central = 7 colonnes (58%), contenant une ligne avec 2 colonnes d'enfants
  • Sidebar droit = 2 colonnes (17%)
  • Démontre le concept de "nesting" (imbrication) en Bootstrap

Styles Appliqués

  • Sidebars : Fond gris standard
  • Content-top : Fond blanc
  • Content-bottom : Fonds alternés pour la distinction
  • Responsive : Container-fluid pour meilleure adaptabilité

Cas d'Usage

Parfait pour :

  • Les dashboards complexes
  • Les pages avec sidebars double et contenu subdivisé
  • Les applications web modernes
  • Les interfaces avec plusieurs niveaux de hiérarchie
  • La démonstration de nesting Bootstrap avancé
Explication - Layout 7 (Layout convertible)

Structure du Layout 7

Un layout responsive qui se convertit automatiquement entre 2 colonnes (écrans larges) et 1 colonne (écrans mobiles).

Composants

  • Header : Banneau supérieur (col-12)
  • Introduction : Section de présentation (col-12)
  • Deux Sections : Deux colonnes qui s'empilent sur mobile
    • Section 1 (col-6)
    • Section 2 (col-6, fond différent)
  • Footer : Pied de page (col-12)

Système de Grille

Responsive 6/6 conversion :

  • Sur grand écran : Deux colonnes côte à côte (50% chacune)
  • Sur petit écran : Une colonne pleine largeur (100%)
  • Utilise col-6 sans point de rupture spécifique = 50% toujours
  • Le container par défaut s'adapte à la taille de l'écran

Styles Appliqués

  • Header/Footer : Couleurs sombres
  • Intro : Fond distinct
  • Sections : Fonds alternés pour clarté visuelle
  • Tous éléments : Hauteurs minimales pour démonstration

Cas d'Usage

Idéal pour :

  • Les pages d'accueil simples
  • Les landing pages mobiles first
  • Les contenus qui s'adaptent bien sur tous appareils
  • La démonstration de principes responsive basiques
  • Les sites nécessitant une simplicité d'adaptation
Explication - Layout 8 (3 colonnes convertible)

Structure du Layout 8

Un layout responsive avancé qui convertit les colonnes sur différents points de rupture : 2 colonnes en mobile, 3 en tablette, et arrangement spécial en desktop.

Composants

  • Header : Banneau supérieur
  • Introduction : Section de présentation
  • Quatre Sections : Contenu adaptatif
    • Sections 1-3 : col-md-6 col-lg-4 col-xl-3
    • Section 4 : col-md-12 col-lg-12 col-xl-3
  • Footer : Pied de page

Système de Grille

Responsive multi-breakpoint :

  • Mobile (<768px) : col-12 = 1 colonne pleine largeur
  • Tablette (≥768px) : col-md-6 = 2 colonnes (50%)
  • Desktop (≥992px) : col-lg-4 = 3 colonnes (33,33%)
  • Grand écran (≥1200px) : col-xl-3 = 4 colonnes (25%)
  • Section 4 spéciale : Pleine largeur jusqu'en XL, puis réduite

Styles Appliqués

  • Sections : Fonds alternés (#dbdfe5 et #b4bac0)
  • Adaptabilité : Min-height pour démonstration clear
  • Responsive : Pas de scroll horizontal nécessaire

Cas d'Usage

Parfait pour :

  • Les galeries de produits adaptatifs
  • Les portefeuilles/portfolios responsifs
  • Les grilles de contenu multi-breakpoint
  • Les pages avec arrangements différents selon l'appareil
  • Les démonstrations Bootstrap avancées multi-points de rupture
Explication - Layout 9 (4 colonnes convertible)

Structure du Layout 9

Le layout le plus complexe avec conversion sur 4 points de rupture différents : 1 colonne en mobile, 2 en small, 3 en medium, et 4 en grand écran.

Composants

  • Header : Banneau supérieur
  • Introduction : Section de présentation
  • Quatre Sections : Adaptation maximale
    • Premières 3 : col-md-6 col-lg-4 col-xl-3
    • Quatrième : col-md-6 col-lg-12 col-xl-3
  • Footer : Pied de page

Système de Grille

Responsive ultra-avancé sur 4 breakpoints :

  • Mobile (<576px) : col-12 = 1 colonne (100%)
  • Tablette petit (≥576px) : col-sm-6 = 2 colonnes (50%)
  • Tablette (≥768px) : col-md-6 = 2 colonnes (50%)
  • Desktop (≥992px) : col-lg-4 = 3 colonnes (33,33%)
  • Grand écran (≥1200px) : col-xl-3 = 4 colonnes (25%)
  • Cas spécial : Section 4 se réorganise pour rester équilibrée (lg-12)

Styles Appliqués

  • Fonds alternés pour distinction visuelle
  • Clearfix utilisé pour les sauts de ligne responsives
  • Hauteurs minimales pour démonstration claire
  • Espacement adaptatif en fonction de la grille

Cas d'Usage

Idéal pour :

  • Les galeries de produits hautement responsives
  • Les grilles d'images/vidéos adaptatifs
  • Les portefeuilles modernes multi-appareils
  • Les pages avec adaptation complète à tous les écrans
  • Les démonstrations Bootstrap la plus avancées
  • Les applications web profondément responsives
