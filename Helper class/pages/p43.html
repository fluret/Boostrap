Display Table Classes

Use the d-table, d-table-row, and d-table-cell class to force an element to behave like a table, tr and td element, respectively. You can also add breakpoints (sm, md, lg, xl, and xxl) to these classes (d-*-table, d-*-table-row and d-*-table-cell) to control when the element should behave like a table, row or column based on the screen width.
